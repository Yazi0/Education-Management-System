3. Functional Flow Overview
Student Registration

Owner adds student

System generates barcode

Student receives unique Register Number

Class Attendance

Owner scans barcode

System shows student details

Attendance marked automatically

SMS sent to parent/student

Payments

Owner marks payment (or auto-marks after POS integration)

Payment status updated per subject

SMS sent

Dashboard incomes updated

4. Income Formula
Per Subject Income
subject_income = students_paid * subject_fee
owner_income = subject_income * 0.20
teacher_income = subject_income * 0.80
